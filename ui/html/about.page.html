Welcome to Louie's Library!

Louie's Library is a site for sharing ebooks. You can make a request for new items, review books you have read, keep track of your reading progress, and much more. The site is in its early stages, with things around the site changing every day.

Here are a list of things on the roadmap...
  • Collections (setting year of completion, reading goals)
  • Site Announcements
  • Mobi to EPub/PDF Conversions
  • How-To Videos


If you are interested in the development of the site, you can follow its progress on GitHub.

If you have any questions or comments, please send me something directly using the on-site messaging system. (click on someones user profile to start a conversation)
